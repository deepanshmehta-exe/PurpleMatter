Purple Matter is a modern clothing website that allows users to browse and order apparel directly through WhatsApp. The project focuses on simplicity and ease of use, offering a smooth shopping experience with a stylish design.

## Features
- Clean and responsive design built with **HTML** and **CSS**  
- Integrated **WhatsApp ordering** for quick and direct communication  
- Lightweight and user-friendly interface  
- A small amount of **JavaScript** for interactive elements  

## Purpose
This project demonstrates how a minimal front-end stack can power an effective small-scale e-commerce website, especially for local or small businesses.

## Technologies Used
- HTML for structure  
- CSS for styling and layout  
- JavaScript for minor interactivity  

## How to Use
1. Open the project folder and run the `index.html` file in your browser.  
2. Browse through the clothing collections.  
3. Click the WhatsApp order button to place your order directly.
